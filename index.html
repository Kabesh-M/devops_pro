End-to-End DevOps Workflow

This project demonstrates a full delivery lifecycle: source in GitHub, CI/CD with Jenkins, containerization with Docker, deployment to Kubernetes, infrastructure provisioning through Terraform, and observability through Prometheus and Grafana.

GitHub Repository

Versioned source code, IaC, manifests, and dashboard assets in one project.

Jenkins Pipeline

Build, test, image publishing, infra apply, cluster deploy, and monitoring setup stages.

Docker

Nginx-based container image optimized for serving the static production webpage.

Kubernetes

Deployment, service, ingress, and namespace definitions for production rollout.

Terraform

Infrastructure modules and variables for repeatable environment provisioning.

Monitoring

Prometheus scraping and Grafana dashboard provisioning for live visibility.

System status: pipeline assets generated and ready for deployment.
